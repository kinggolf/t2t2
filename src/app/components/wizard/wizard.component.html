Create multiple lists of items with t2t2.

Lists can be anything, e.g. projects, notes, reminders, trips, days, ...

Select menu button at top right to create a new list.

Open a list if it has at least one item by tapping the top of the list, e.g. on the list name.

List items, e.g. tasks, reminders, thoughts, ..., can be added to a list by tapping the add icon on the list.

Edit the list name or title by tapping the edit icon.

Delete a list by tapping the delete icon. You will be asked to confirm before a list is deleted.

If you have multiple lists, then you can reorder them by grabbing/moving the drag and drop icon.

On an open list you can edit item name & description, delete an item, reorder the list of items, and mark complete.

Only one list can be open at a time.
